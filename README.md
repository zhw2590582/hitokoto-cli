# hitokoto-cli
> A Hitokoto Cli Tool

## Install

```
$ npm install --global hitokoto-cli
```

## Usage

```
$ hitokoto-cli --help

    Show random hitokoto
      $ hitokoto-cli

    Show configurable hitokoto
      $ hitokoto-cli -o
    
    Show current version
      $ hitokoto-cli -v

    Source code of this side project
      $ hitokoto-cli -s

    Show help
      $ hitokoto-cli -h
```
